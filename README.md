# webrx scrapers

# Installation
```bash
git clone git@github.com:imega-webrx/crawler.git && cd crawler/
python3 -m venv .venv
. .venv/bin/activate

# run scraper
# change ${SCRAPER_NAME} to required value from list of scrapers
python src/manage.py ${SCRAPER_NAME}
```
